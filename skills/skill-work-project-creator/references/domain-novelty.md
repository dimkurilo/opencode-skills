# domain_novelty

| Value | H-panel | Self-assessment | Platform study |
|-------|---------|-----------------|----------------|
| **FIRST_IN_PORTFOLIO** | Default **ON** (3–5 falsifiable Hs) | ON for ≥10% roles | ON if harness unknown |
| **REPEAT_DOMAIN** | Default **OFF** | Only if roster changes | OFF if MEMORY has notes |
| **TRANSFER** | Optional ≤3 Hs | ON | Selective |

## How to set

1. Search portfolio MEMORY / prior programs for same domain.
2. If first time multi-agent in this domain → FIRST.
3. If same domain OS already proven → REPEAT (skip H-panel).
4. If adjacent domain with partial transfer → TRANSFER.

Record value in SPEC.md and MEMORY.md LEARNED_RULES.
